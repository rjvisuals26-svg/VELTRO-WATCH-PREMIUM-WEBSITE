## 📖 About

Veltro is a frontend web development project aimed at creating a high-end digital boutique for luxury watches. The design focuses on conveying sophistication, mechanical precision, and uncompromising craftsmanship through a cinematic user experience.

---

## ✨ Features

- **Cinematic Hero Section:** Engaging video backgrounds with smooth reveal animations.
- **Interactive Product Showcase:** Detailed watch cards with hover effects and quick view functionality.
- **GSAP Animations:** Smooth scroll and reveal animations powered by ScrollTrigger and Lenis.
- **Fully Responsive:** Optimized layout for desktop, tablet, and mobile devices.
- **Elegant Typography:** Utilizes premium fonts (Cormorant Garamond and Inter) for a luxurious feel.

---

## 🛠️ Built With

- **HTML5:** Semantic markup structure.
- **Vanilla CSS:** Custom styling, CSS variables for theming, and responsive design.
- **JavaScript:** DOM manipulation and interactive elements.
- **[GSAP](https://gsap.com/):** Professional-grade JavaScript animation library.
- **[Lenis](https://lenis.darkroom.engineering/):** Smooth scrolling library for web.
