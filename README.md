## 📖 About

Veltro is a frontend web development project aimed at creating a high-end digital boutique for luxury watches. The design focuses on conveying sophistication, mechanical precision, and uncompromising craftsmanship through a cinematic user experience.

---

## ✨ Features

- **Cinematic Hero Section:** Engaging media environments featuring premium typography layouts with smooth, calculated reveal transitions.
- **Interactive Product Showcase:** Detailed watch cards utilizing high-contrast styling details, clean hover properties, and structured layouts.
- **Advanced Dynamic Interactions:** Implements smooth scroll mechanics and directional viewport reveal triggers utilizing **GSAP ScrollTrigger** and **Lenis**.
- **Responsive Fluid Architecture:** Tailored viewing profiles perfectly formatted across desktop layouts down to touch-friendly mobile display Viewports.
- **Editorial Typography Integration:** Melds premium editorial serifs (*Cormorant Garamond*) with technical modern sans-serif fonts (*Inter*) to balance luxury history with technical design.

---

## 🛠️ Built With

*   **HTML5:** Semantic frontend architecture.
*   **Vanilla CSS3:** Native structural layout engineering utilizing custom component parameters and responsive design queries.
*   **JavaScript (ES6+):** Programmatic DOM interactivity controls.
*   **[GSAP (GreenSock)](https://gsap.com/):** High-performance UI timeline sequence animations.
*   **[Lenis Smooth Scroll](https://lenis.darkroom.engineering/):** Viewport acceleration management framework.
